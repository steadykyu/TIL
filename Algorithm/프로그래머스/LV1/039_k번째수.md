```java
public int[] solution(int[] array, int[][] commands) {
        int[] answer = new int[commands.length];

        for(int p = 0 ; p < commands.length; p++){
            // 1. i, j , k 읽어오기 ex) 2, 5, 3
            int i = commands[p][0];
            int j = commands[p][1];
            int k = commands[p][2];

            // 2. 배열 자르고 정렬하기 1 4
            int[] array2 = new int[j-i+1];
            for(int x = 0; x < array2.length; x++){
                array2[x] = array[x+i-1];
            }

            Arrays.sort(array2);

            // 3. k 번쨰 숫자 가져오고 answer에 넣어주기
            answer[p] = array2[k-1];
        }

        return answer;
    }
```


## 참고

> 이차원배열과 Arrays.copyOfRange() 활용하기.
```java
import java.util.Arrays;
class Solution {
    public int[] solution(int[] array, int[][] commands) {
        int[] answer = new int[commands.length];

        for(int i=0; i<commands.length; i++){
            int[] temp = Arrays.copyOfRange(array, commands[i][0]-1, commands[i][1]);
            Arrays.sort(temp);
            answer[i] = temp[commands[i][2]-1];
        }

        return answer;
    }
}
```
1. Arrays.copyOfRange(int[] original, int from, int to) 를 활용하면 더 쉽게 접근 할 수 있다. <=  < 의 관계
